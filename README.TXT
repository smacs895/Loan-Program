PROJECT TITLE: 
Loan

PURPOSE OF PROJECT:
Program to calculate a loan with a loanAmount, numberOfYears for the loan and an annualInterestRate.

VERSION or DATE:
Spring 2019

HOW TO START THIS PROJECT:
Driver program will create 4 loans. 2 with the defaults for loanAmount, numberOfYears and annualInterestRate. Loan number 3 and 4 will be created with different values.
